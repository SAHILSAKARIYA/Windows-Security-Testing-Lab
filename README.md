# Windows-Security-Testing-Lab

## Overview
This project demonstrates a controlled exploitation simulation using Kali Linux and Windows 10 in an isolated virtual lab environment. The objective was to understand reverse shell payload generation and basic post-exploitation techniques for cybersecurity learning purposes.

## Objectives
- Build a secure virtual lab environment
- Generate a reverse TCP payload
- Establish a Meterpreter session
- Perform basic post-exploitation activities
- Understand attacker methodologies in a safe environment

## Lab Environment
- Attacker Machine: Kali Linux
- Target Machine: Windows 10
- Virtualization Platform: VMware

## Tasks Performed
- Configured isolated VMware lab environment
- Generated reverse TCP payload using msfvenom
- Hosted payload using Python HTTP Server
- Established Meterpreter session using Metasploit Framework
- Performed system enumeration
- Captured remote screenshots for demonstration

## Tools & Technologies
- Kali Linux
- Windows 10
- Metasploit Framework
- msfvenom
- VMware
- Python HTTP Server

## Skills Learned
- Ethical Hacking Fundamentals
- Exploitation Techniques
- Payload Generation
- Meterpreter Basics
- Post-Exploitation Enumeration
- Virtual Lab Setup

## Disclaimer
This project was conducted in a private and isolated lab environment for educational and ethical cybersecurity training purposes only.
